# DocuMind Backend

This is the FastAPI backend for the Local AI Document Assistant (DocuMind).

## Folder Structure

- `app/`: Contains the FastAPI application logic and core integration (`main.py`, `rag.py`, `embeddings.py`, `database.py`)
- `services/`: Contains API service interfaces (Ollama, Chroma, etc.)
- `models/`: Contains the Pydantic type definitions and data models
